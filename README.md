# 🚀 Financial Intelligence Platform

An AI-powered financial dashboard built using React, Firebase, Tailwind CSS, and modern web technologies.

## 🌟 Features

* 🔐 Firebase Authentication (Email & Google Login)
* 📊 Interactive Financial Dashboard
* 📈 Stock Analysis Section
* 🤖 AI-Based Predictions
* 📰 News Sentiment Analysis
* ⭐ Watchlist Management
* 💼 Portfolio Tracking
* 🚨 Alerts System
* ⚙️ User Settings
* 🌙 Modern Dark UI Design
* 📱 Responsive Layout

## 🛠️ Tech Stack

* React.js
* Vite
* Tailwind CSS
* Firebase Authentication
* Firestore Database
* React Router DOM
* Lucide React Icons

## ⚡ Installation

Clone the repository:

```bash
git clone https://github.com/indarsinghrajawt/finance-platform.git
```

Go to project folder:

```bash
cd finance-platform
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

## 🔑 Firebase Setup

Create a Firebase project and add your configuration inside:

```bash
src/firebase.js
```

Enable:

* Email/Password Authentication
* Google Authentication
* Firestore Database

Add your deployment domain in:

Authentication → Settings → Authorized Domains
